Lesson 3: About CSS

Cascading rules: start with least specific and finish with very specific rules.

Ways to include CSS styling in webpages:

  1. write CSS into the head of your HTML
    - recommended to small projects
    - you have to include it into a style element
  2. link your HTML to a separate CSS file
    - for larger projects
    - in your HTML doc. create the head element and add a link tag inside:

  3. write your style inline in your HTML code.
    - not recommended (confusing and unprofessional)

Comments in your code

It is a thing what makes you a good programmer. By using comments not only you, but any other programmer will be able to quickly understand what, how and why did you code the certain things. In the comments you can justify and clearly describe the different parts of your code.

Difference between div and span

div is a block level element (for a whole block)
span is an inline element, we use it for text decoration or simple inline styling

Difference between id and class

id identifies an element, it's unique
class can be used for more elements, helps to avoid repetition (often used)

Code, Test, Refine

  1. look for natural boxes
  2. look for repeated elements, semantic elements
  3. write your HTML
  4. apply your CSS styles (from the biggest to the smallest details)
  5. fix (small things)
