Lesson 1: World Wide Web

World Wide Web (www) is a collection of HTML documents - HyperText Markup Language. The following picture is an illustration how the web is built up of its components.

When you want to open a website you need client to do so - computer and browser. The browser sends request via the internet to servers. These requests using the so called HTTP which is a protocol for communication. The server responds by sending files (HTML) that your browser displays.

HTML

It is made up of:

  • text content - what you can see
  • markup - what it looks like
  • references to other documents - like images or videos
  • links to other pages - url

Markup

The markup is made up of tags. These tags will help you to build a website. When you want to add something to your website, you have to start with an opening tab and after you can add the desired content. When you typed that content you have to finish it with a closing tag.

With the following tags you can format your text. There are more formatting tags, these are basic examples.
b = bold - it makes your text bold.
em = emphasis - it makes your text italic.

HTML Attributes

For some tags you can specify attributes which provide additional information about an element, we call it value. These are always given in the opening tags. Let's see examples.

Links and Images

You can use attributes for links and images.

a = anchor - we use this tag to create a link
In the opening tag right after the "a" you have to write the attribute, href(which refers to the link)="and write/copy and paste the url link here." Then you have to close the opening tag. In the following step you have to define the content what you want to see on the website, for eg. "Slippity Sup" which is going to appear on the screen. By clicking on this, the computer will open that wepsite which you provided in the reference area (href=" "). Finish it with a closing tag.


igm = image - with this tag you can add images to a website
In the opening tag right after the "img" you have to write the attribute, src(source)="write/copy and paste here the url link or the file name which exists on your computer", here you can add another attribute alt(alternate)=" ". If you don't add this, it won't break your code. However this is nice to have in case of a broken request or could be helpful for blind.

Whitespace

No matter how much space or enter you use between sentences or lines, it won't appear in the browser, you can only see it in your text editor. If you are willing to make some extra space you have two tags to choose from.

br = break (void) - this will make a line break
It is an inline tag which means that it only covers the space between the opening tag and the closing tag.

p = paragraph - this makes an individual paragraph
This is a block tag meaning that is bounded by an invisible box which filled by space.

For further differentiation of break and paragraph, please see this video.

Framework of HTML document

At the first place always start with !DOCTYPE html. By giving this you will identify the type of the document.

!DOCTYPE html

html - opening tag

head - opening tag
title - opening and closing tag
head - closing tag

body - opening tag
contents (could be different kind of tags (eg.: h1, p,)
body - closing tag

html - closing tag
