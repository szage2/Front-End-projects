Lesson 2: Structure of HTML

Tree structure

With other word it is DOM (Document Object Model) which is used in HTML to create a clear structure. It reminds us for a family tree. With this structure we are able to identify different sections, or differentiate "boxes". In the below picture, you can see how it is looks like.

HTML

HTML is a structure using a language of the web which consists of syntax and different rules. This structure consists of elements with an opening tag and a closing tag. These Tags are build up based on the DOM.

CSS

CSS connects with HTML, because this is using a language (syntax and rules) for creating the style for your website. With CSS you are able to color and design the background, and other things according to your taste (or the requirements of your employer).

Boxes

All of the websites are building up of invisible boxes. You use these boxes to identify the different sections of the website. You make a box for the head, body, footer, but you can tear down these to further sections with boxes. Each box is an element. The following picture illustrates these boxes.
